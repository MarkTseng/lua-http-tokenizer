About
=====

A HTTP protocol tokenizer.


Installation
============

Lastest Git Revision
--------------------

With CMake:

	$ git clone git://github.com/Neopallium/lua-http-tokenizer.git
	$ cd lua-http-tokenizer ; mkdir build ; cd build
	$ cmake ..
	$ make
	$ sudo make install


Example usage
=============

	local http_tokenizer = require"http_tokenizer"
